# ğŸ—ï¸ Project Architecture

The architecture of the **Develop Guard India Pvt. Ltd. Website** is designed to be simple, efficient, and scalable. It follows a layered approach, separating the frontend, backend, and deployment environments. Below is an overview of the project's architecture:

The frontend is responsible for the user interface and user experience. It is built using:
- **HTML**: For structuring content.

--

## **2. Backend Layer**
The backend handles the logic, data processing, and server-side operations. It uses:
- **Flask**: A micro web framework for routing and processing requests.
- **SMTP**: For sending emails directly from the website.

--

## **3. Deployment Layer**
The deployment layer ensures that the website is hosted and accessible online. It includes:
- **Hostinger VPS**: The hosting service for deploying the website.

Version control ensures collaboration and tracking of changes in the codebase:
- **Git**: Used for source code management.

The development and testing environment is configured as follows:
- **Local OS**: Windows, used by developers for coding and testing.

## **6. Workflow**
1. **Frontend**: Users interact with the frontend, which sends requests to the backend.

# ğŸ› ï¸ Installation and Setup

Follow the steps below to set up and run the **Develop Guard India Pvt. Ltd. Website** project on your local machine or VPS server.

---

## **1. Clone the Project**

First, clone the repository from GitHub:

```bash
git clone https://github.com/Beginning-co-in/BSS03
```

Change to the project directory:

```bash
cd BSS03
```

---

## **2. Running Locally**

To run the project locally on your system, follow these steps:

### Step 1: Start a Virtual Environment
Create and activate a virtual environment:

```bash
# On Windows
python -m venv venv
venv\Scripts\activate

# On Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### Step 2: Install Dependencies
Install the required Python packages:

```bash
pip install -r requirements.txt
```

### Step 3: Run the Application
Run the Flask application:

```bash
python main.py
```

You will see a message like this:

```
 * Running on http://127.0.0.1:5050
```

Open the URL in your browser to view the website.

---

## **3. Running on VPS Server**

To run the project on the VPS server, follow these steps:

### Step 1: Navigate to the Project Directory
Change to the project directory on the VPS:

```bash
cd /var/www/BSS03
```

### Step 2: Start a Virtual Environment
Activate the virtual environment:

```bash
source venv/bin/activate
```

### Step 3: Run the Application
Run the Flask application:

```bash
python main.py
```

The application will start, and you will see a message like this:

```
 * Running on http://127.0.0.1:5050
```

The website will now be accessible on the configured VPS server.
